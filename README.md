# Srinadhu Krishna Prabath Portfolio

Static portfolio site built from the provided resume PDF and public LinkedIn profile.

Double-click `Open-Portfolio.bat` to start the local server and open the portfolio.

You can also open `index.html` directly in a browser, or run:

```powershell
node server.js
```

Then visit `http://127.0.0.1:8080`.

## Included

- `index.html` - portfolio page
- `styles.css` - responsive styling
- `script.js` - small scroll reveal and footer year script
- `assets/analytics-hero.png` - generated hero visual
- `assets/Srinadhu-Krishna-Prabath-Updated-Resume.pdf` - updated resume PDF
- `resume-updated.html` - editable/printable resume source
